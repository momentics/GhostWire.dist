# GhostWire.dist

Public distribution repository for GhostWire binary releases.

This repository does not contain the protected source code.
It is used only for:

- GitHub Releases with prebuilt binary artifacts
- public install metadata
- compatibility and platform documentation

## Repository Layout

```text
.
├── README.md
├── docs/
│   ├── abi-policy.md
│   └── platform-matrix.md
├── install/
│   ├── fetch.ps1
│   └── fetch.sh
└── manifests/
```
